deleteGroups - deletes a list of groups from the active network

USAGE

    deleteGroups <group-list>

DESCRIPTION

Given a * this deletes all groups except the BIAS group from the active network. Otherwise it deletes those groups listed. Any remaining groups will be renumbered to reflect their new positions in the group array.
