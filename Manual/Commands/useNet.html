useNet - activates a network or prints a list of all networks

USAGE

    useNet [<network>]

DESCRIPTION

With no argument, this returns a list of all loaded networks. Given the name of a network, it makes that the active, or current network.

EXAMPLES

    lens> useNet
    "bar" "foo"
    lens> useNet foo
