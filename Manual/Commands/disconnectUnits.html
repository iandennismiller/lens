disconnectUnits - deletes links of a specified type between two units

USAGE

    disconnectUnits <unit1> <unit2> [-type <link-type>]

DESCRIPTION

This removes the link of the specified type that projects from unit1 to unit2. If no type is given, all links are removed.

EXAMPLES

To remove the link from the bias unit to "hidden:3": 

    lens> disconnectUnits bias:0 hidden:3
