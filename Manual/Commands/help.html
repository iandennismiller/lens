help - prints the command summaries or explains a command

USAGE

    help [(-s | <command>)]

DESCRIPTION

With no arguments, help will display a list of commands and their synopses organized by topic.

The argument -s causes help to print a sorted list of commands and their synopses.

Given a command, help will display a manual page about the command or report whether it is a valid alias.
