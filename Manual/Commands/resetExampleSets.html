resetExampleSets - returns the example set to the first example

USAGE

    resetExampleSets <example-set-list>

DESCRIPTION

This resets the example sets to their initial state. In ORDERED mode this means the next example presented will be the first. In PERMUTED mode it restarts the permutation. It has no effect in RANDOMIZED, PROBABILISTIC, PIPE, or CUSTOM modes.
