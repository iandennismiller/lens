deleteLinks - deletes all links of a specified type

USAGE

    deleteLinks [-type <link-type>]

DESCRIPTION

This removes all links of the specified type. If no type is given, all links are removed.

EXAMPLES

To remove all bias links: 

    lens> deleteAllLinks -type bias
