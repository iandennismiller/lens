resetDerivs - clears the unit and link error derivatives

USAGE

    resetDerivs

DESCRIPTION

Sets the unit input, output, and gain derivatives to zero and sets the link weight derivatives to zero. The updateWeights command will also do this, but only after using the derivatives to update the link weights.

SEE ALSO
